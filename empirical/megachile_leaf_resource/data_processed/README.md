# Generated evidence outputs

This directory contains outputs produced from a reviewed evidence ledger:

- `host_evidence_validated.csv`: original ledger rows plus fatal validation errors and warnings;
- `primary_host_universe.csv`: one evidence-backed row per plant taxon eligible for the first Japanese material-use analysis.

These outputs are ignored by default because they are regenerated from the curated ledger and protocol. Commit a versioned snapshot only when it is ready to support a release or manuscript analysis.
