# Broad evidence to fixed regime interface

## Decision

The mathematics of the attraction--defence model is retained unchanged. The exact
local interaction condition remains:

```text
∂²W/∂A∂D = H*d_A*e_F - P*b_A*c_D*exp(-c_D*D)*(1-c_R*R) - c_AD
```

The broad L1/L2 literature analysis is not used to fit this equation. It instead
maps which empirical channels supply a defensible sign/context constraint for each
term, and which terms must remain explicit sensitivity axes.

## Why the broad analysis changes

The strict route registry is too small to be the sole empirical analysis. The
fixed broad corpus is large enough to support a shallow evidence map, provided
its claim is kept at the appropriate level.

```text
L1: all fixed retrieved records
L2: metadata-enriched biological candidate layer
Abstract map: broad co-mention candidates in accessible Crossref abstracts
Direction registry: source-adjudicated, low-count sign/context anchors
Regime model: unchanged exact expression and parameter sweep
```

The abstract map does not replace source coding. It tells us where the literature
has organ-matched attention to an edge; the direction registry tells us whether
there is any source-coded directional support for the branch used in the model.

## Parameter correspondence

| Model term | Needed empirical channel | What broad mapping can say | What it cannot say |
|---|---|---|---|
| `b_A` | A → pollination | whether floral-attraction/pollination abstracts are dense; whether source anchors include a positive or mixed direction | magnitude of attraction-mediated outcross gain |
| `d_A` | A → floral antagonism | whether attraction/florivory abstracts exist and whether display tracking is source-observed | a universal positive tracking coefficient |
| `e_F` | B → floral antagonism | whether floral barriers and antagonism are studied together; whether reduced antagonism is source-observed | barrier efficacy for every defence type |
| `c_D` | B → pollination | whether defence/access and pollination are studied together; whether negative pollinator-use anchors occur | a universal pollinator cost or its magnitude |
| `c_AD` | matched A+B allocation/cost | only whether shared-cost language appears in broad abstracts | a shared investment cost; this needs matched trait data |
| `c_R*R` | reproductive assurance/outcross dilution | nothing in the active L1/L2 program | any calibration; retain as sensitivity only |

## Allowed use in the model

The broad evidence map can partition model scenarios into three classes:

1. **structurally admissible and directionally anchored**: a model branch has at
   least some source-coded support for its sign in a defined trait/outcome context;
2. **structurally admissible but condition-dependent**: source anchors are mixed,
   so the sign remains a scenario rather than a global statement;
3. **unidentified sensitivity axis**: the term is mathematically necessary but has
   no matched empirical module in the current literature corpus.

It must not:

- set a numerical value of any model parameter from publication counts;
- treat abstract co-mention as a causal effect;
- infer an A--B covariance from separate A/P and B/H literatures;
- use leaf- or stem-defence evidence as a floral-barrier test without an explicit
  biological bridge.

## Current interpretation

The retained model already says that complementarity is possible when display
tracking by floral antagonists and floral-defence efficacy outweigh pollinator
obstruction and shared investment cost; substitution occurs in the opposite
regime. The broad analysis makes this claim empirically legible without
pretending that its parameters are already calibrated.

The direct B→P anchors are therefore not the complete empirical analysis. They
serve a narrower role: they keep the `c_D > 0` branch biologically credible for
some chemical defence/access contexts, while the broad map establishes how sparse
or common that edge is in the fixed literature architecture.
